Jumbotron

Jumbotron, é uma caixa cinza com os cantos arredondados, que vai de um lado ao outro da página, se você quer que o mesmo não vá de uma lado ao outro, colocar dentro de uma div com a classe de container.

Obs: o memso está compordas, pois é uma configuração do css da página
